# UI Package

A collection of reusable React components.

## Installation

```bash
npm install @camplight-task/ui
```

## Available Components

- Alert
- Button
- Card
- Dialog
- ErrorBanner
- Input
- Label
- LoadingGrid
- SearchAndFilter
- Select
- Skeleton
- Textarea

## Usage

Import components from the package:

```jsx
import { Button, Card } from '@camplight-task/ui'
```

## Development

Run linting:
```bash
npm run lint
```

Check types:
```bash
npm run check-types
```
